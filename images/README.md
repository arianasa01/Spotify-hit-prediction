# Visual outputs

This folder contains selected visual outputs from the original notebook. The raw Kaggle dataset is not included in the repository, so these images help show the analysis results without requiring the full data file to be uploaded.

The figures include:

- correlation and popularity distribution plots
- country-level word clouds
- sentiment density plots
- geographic audio-feature maps
- seasonal audio-feature trends
- regression model comparison plots
- classification model results and feature importance

The same figures can be regenerated by downloading the Kaggle dataset and running the Python workflow or the notebook.
