# Data folder

The dataset is not included in this repository because the raw CSV file is large.

Download it from Kaggle:

https://www.kaggle.com/datasets/asaniczka/top-spotify-songs-in-73-countries-daily-updated

Then place the file in this folder path:

```text
data/raw/universal_top_spotify_songs.csv
```

Expected file name:

```text
universal_top_spotify_songs.csv
```

After running the project, the cleaned monthly dataset will be created automatically here:

```text
data/processed/spotify_monthly_2024.csv
```

The original coursework used a snapshot downloaded in November 2024. Since the Kaggle dataset is updated over time, your results may be slightly different if you download a newer version.
