# Fixing the GitHub upload

The previous upload has files with broken line breaks. Replace the repository contents with this cleaned version.

## Best method using GitHub Desktop

1. Open GitHub Desktop.
2. Open your `Spotify-hit-prediction` repository.
3. Click `Repository` > `Show in Explorer`.
4. Delete the current incorrect files and folders, except `.git` if you can see it.
5. Copy all files and folders from this cleaned package into the repository folder.
6. Go back to GitHub Desktop.
7. In the summary box, write:

```text
Fix README formatting and Python file structure
```

8. Click `Commit to main`.
9. Click `Push origin`.

After refreshing GitHub, the README should show proper headings, paragraphs and code blocks. The Python files should also have normal line breaks and indentation.
